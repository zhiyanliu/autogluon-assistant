## Usage

MTEB (Massive Text Embedding Benchmark) evaluates text embedding models across diverse NLP tasks and datasets. It uses the BEIR benchmark for retrieval (15 datasets, including 12 CQADupstack subsets).

### Load Model & Define Tasks

```python
import mteb
from sentence_transformers import SentenceTransformer

model = SentenceTransformer("BAAI/bge-base-en-v1.5")

# Full MTEB English retrieval tasks list
retrieval_tasks = [
    "ArguAna", "ClimateFEVER", "CQADupstackAndroidRetrieval",
    "CQADupstackEnglishRetrieval", "CQADupstackGamingRetrieval",
    "CQADupstackGisRetrieval", "CQADupstackMathematicaRetrieval",
    "CQADupstackPhysicsRetrieval", "CQADupstackProgrammersRetrieval",
    "CQADupstackStatsRetrieval", "CQADupstackTexRetrieval",
    "CQADupstackUnixRetrieval", "CQADupstackWebmastersRetrieval",
    "CQADupstackWordpressRetrieval", "DBPedia", "FEVER", "FiQA2018",
    "HotpotQA", "MSMARCO", "NFCorpus", "NQ", "QuoraRetrieval",
    "SCIDOCS", "SciFact", "Touche2020", "TRECCOVID",
]
```

### Run Evaluation

```python
tasks = mteb.get_tasks(tasks=["ArguAna"])  # or any subset
evaluation = mteb.MTEB(tasks=tasks)
results = evaluation.run(model, output_folder="results")
```

### Output

Results are saved to `{output_folder}/{model_name}/{model_revision}/{task_name}.json`, containing metrics (map, mrr, ndcg, precision, recall) at cutoffs 1–1000. Key results for `bge-base-en-v1.5` on ArguAna:

| Metric | Score |
|---|---|
| **ndcg@10 (main_score)** | **0.636** |
| map@10 | 0.558 |
| recall@10 | 0.884 |
| mrr@10 | 0.560 |
