# Embedding Model Evaluation on MS Marco

## Setup & Dataset

```python

```

```python
from datasets import load_dataset
import numpy as np

data = load_dataset("namespace-Pt/msmarco", split="dev")

# Truncated dataset: 100 queries, corpus from first 5000 queries' positives
queries = np.array(data[:100]["query"])
corpus = sum(data[:5000]["positive"], [])
```

## Embedding

```python
from FlagEmbedding import FlagModel

model = FlagModel('BAAI/bge-base-en-v1.5',
                  query_instruction_for_retrieval="Represent this sentence for searching relevant passages:",
                  use_fp16=True)

corpus_embeddings = model.encode(corpus)
```

## Faiss Indexing

- `dim`: vector dimension (768 for bge-base-en-v1.5)
- `'Flat'`: exhaustive search
- `faiss.METRIC_INNER_PRODUCT`: inner product distance metric

```python
import faiss

dim = corpus_embeddings.shape[-1]
index = faiss.index_factory(dim, 'Flat', faiss.METRIC_INNER_PRODUCT)
corpus_embeddings = corpus_embeddings.astype(np.float32)
index.train(corpus_embeddings)
index.add(corpus_embeddings)

# Save/load index for reuse:
# faiss.write_index(index, "./index.bin")
# index = faiss.read_index("./index.bin")
```

## Retrieval

```python
query_embeddings = model.encode_queries(queries)
ground_truths = [d["positive"] for d in data]
corpus = np.asarray(corpus)

from tqdm import tqdm

res_scores, res_ids, res_text = [], [], []
cut_offs = [1, 10]
k = max(cut_offs)
batch_size = 256

for i in tqdm(range(0, len(query_embeddings), batch_size), desc="Searching"):
    q_embedding = query_embeddings[i: min(i+batch_size, len(query_embeddings))].astype(np.float32)
    score, idx = index.search(q_embedding, k=k)
    res_scores += list(score)
    res_ids += list(idx)
    res_text += list(corpus[idx])
```

## Evaluation Metrics

### Recall — fraction of relevant results found within top-k

```python
def calc_recall(preds, truths, cutoffs):
    recalls = np.zeros(len(cutoffs))
    for text, truth in zip(preds, truths):
        for i, c in enumerate(cutoffs):
            recall = np.intersect1d(truth, text[:c])
            recalls[i] += len(recall) / max(min(c, len(truth)), 1)
    recalls /= len(preds)
    return recalls
```

### MRR — average reciprocal rank of the first relevant result

```python
def MRR(preds, truth, cutoffs):
    mrr = [0 for _ in range(len(cutoffs))]
    for pred, t in zip(preds, truth):
        for i, c in enumerate(cutoffs):
            for j, p in enumerate(pred):
                if j < c and p in t:
                    mrr[i] += 1/(j+1)
                    break
    mrr = [k/len(preds) for k in mrr]
    return mrr
```

### nDCG — ranking quality considering position and relevance, computed via `sklearn`

```python
# Create binary relevance encodings
pred_hard_encodings = []
for pred, label in zip(res_text, ground_truths):
    pred_hard_encodings.append(list(np.isin(pred, label).astype(int)))

from sklearn.metrics import ndcg_score
for i, c in enumerate(cut_offs):
    print(f"nDCG@{c}: {ndcg_score(pred_hard_encodings, res_scores, k=c)}")
```

**Key points:**
- **Embeddings must be cast to `np.float32`** before Faiss indexing/searching
- Use `encode_queries()` (not `encode()`) for queries — applies the retrieval instruction prefix
- Save Faiss index to avoid re-encoding large corpora
- `cut_offs` controls evaluation granularity (e.g., `[1, 10]` for @1 and @10)
