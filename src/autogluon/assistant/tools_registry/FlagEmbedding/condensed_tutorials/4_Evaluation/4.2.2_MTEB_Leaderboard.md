## Running Full Evaluation

The 7 task types: **Classification** (F1), **Clustering** (v-measure), **Pair Classification** (avg precision), **Reranking** (MRR@k/MAP), **Retrieval** (nDCG@k via BEIR), **STS** (Spearman/cosine), **Summarization** (Spearman/cosine).

```python
import mteb
from mteb.benchmarks import MTEB_MAIN_EN
from sentence_transformers import SentenceTransformer

# Load model (either method works)
model = SentenceTransformer("BAAI/bge-base-en-v1.5")
# or: model = mteb.get_model("BAAI/bge-base-en-v1.5")

# Evaluate all 56 datasets
for task in MTEB_MAIN_EN.tasks:
    eval_splits = ["dev"] if task == "MSMARCO" else ["test"]
    evaluation = mteb.MTEB(tasks=[task], task_langs=["en"])
    evaluation.run(model, output_folder="results", eval_splits=eval_splits)
```

## Submitting to Leaderboard

Results are stored in `results/{model_name}/{model_revision}`. Generate metadata:

```bash
mteb create_meta --results_folder results/{model_name}/{model_revision} --output_path model_card.md
# If README already exists:
mteb create_meta --results_folder results/{model_name}/{model_revision} --output_path model_card.md --from_existing your_existing_readme.md
```

Copy `model_card.md` contents to the top of your model's HF Hub README.md. The leaderboard refreshes daily.

## Partial Evaluation

You can submit results for specific task categories only. Example for clustering:

```python
TASK_LIST_CLUSTERING = [
    "ArxivClusteringP2P", "ArxivClusteringS2S",
    "BiorxivClusteringP2P", "BiorxivClusteringS2S",
    "MedrxivClusteringP2P", "MedrxivClusteringS2S",
    "RedditClustering", "RedditClusteringP2P",
    "StackExchangeClustering", "StackExchangeClusteringP2P",
    "TwentyNewsgroupsClustering",
]

evaluation = mteb.MTEB(tasks=TASK_LIST_CLUSTERING)
results = evaluation.run(model, output_folder="results")
```

Then repeat the submission steps — your model appears in the relevant leaderboard section.

**Note:** MTEB is developing an updated, more concise English benchmark. Check their [GitHub](https://github.com/embeddings-benchmark/mteb) for updates.
