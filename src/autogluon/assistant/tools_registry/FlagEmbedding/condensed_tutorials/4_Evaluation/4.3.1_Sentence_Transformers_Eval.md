## Retrieval Evaluation with BeIR

Uses `InformationRetrievalEvaluator` against the BeIR Quora dataset.

```python
import random
from sentence_transformers.evaluation import InformationRetrievalEvaluator
from datasets import load_dataset

# Load BeIR Quora dataset
corpus = load_dataset("BeIR/quora", "corpus", split="corpus")
queries = load_dataset("BeIR/quora", "queries", split="queries")
relevant_docs_data = load_dataset("BeIR/quora-qrels", split="validation")
```

**Shrink corpus** to relevant documents + 10,000 random docs, then convert to required dict formats:

```python
required_corpus_ids = list(map(str, relevant_docs_data["corpus-id"]))
required_corpus_ids += random.sample(corpus["_id"], k=10_000)
corpus = corpus.filter(lambda x: x["_id"] in required_corpus_ids)

corpus = dict(zip(corpus["_id"], corpus["text"]))       # cid => document
queries = dict(zip(queries["_id"], queries["text"]))     # qid => question
relevant_docs = {}                                        # qid => set([relevant_cids])
for qid, corpus_ids in zip(relevant_docs_data["query-id"], relevant_docs_data["corpus-id"]):
    qid, corpus_ids = str(qid), str(corpus_ids)
    if qid not in relevant_docs:
        relevant_docs[qid] = set()
    relevant_docs[qid].add(corpus_ids)
```

**Run evaluation** — computes standard IR metrics (e.g., NDCG, MAP, recall):

```python
ir_evaluator = InformationRetrievalEvaluator(
    queries=queries,
    corpus=corpus,
    relevant_docs=relevant_docs,
    name="BeIR-quora-dev",
)
results = ir_evaluator(model)
```

**Key details:** `InformationRetrievalEvaluator` expects `queries` and `corpus` as `dict[str, str]`, and `relevant_docs` as `dict[str, set[str]]` mapping query IDs to sets of relevant corpus IDs.
