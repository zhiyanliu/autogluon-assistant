## Datasets

35 datasets across 6 tasks: **Classification** (F1), **Clustering** (v-measure), **Pair Classification** (avg precision), **Reranking** (MRR@k, MAP), **Retrieval** (nDCG@k via BEIR), **STS** (Spearman correlation on cosine similarity).

```python

    'T2Retrieval', 'MMarcoRetrieval', 'DuRetrieval', 'CovidRetrieval', 'CmedqaRetrieval', 'EcomRetrieval', 'MedicalRetrieval', 'VideoRetrieval',

]
```

## Model Loading

**Option 1 — FlagDRESModel** (instruction used for retrieval tasks):
```python
from ...C_MTEB.flag_dres_model import FlagDRESModel

model = FlagDRESModel(model_name_or_path="BAAI/bge-base-zh-v1.5",
                      query_instruction_for_retrieval="为这个句子生成表示以用于检索相关文章：",
                      pooling_method="cls")
```

**Option 2 — SentenceTransformer:**
```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer("PATH_TO_MODEL")
```

**Option 3 — Custom class** (must implement `encode` returning `List[np.ndarray]` or `List[tensor]`):
```python
class MyModel():
    def __init__(self):
        pass
    def encode(self, sentences, batch_size=32, **kwargs):
        pass
```

## Evaluation

**GPU highly recommended** for efficiency.

```python
import mteb
from mteb import MTEB

tasks = mteb.get_tasks(ChineseTaskList)
for task in tasks:
    evaluation = MTEB(tasks=[task])
    evaluation.run(model, output_folder=f"zh_results/{model_name.split('/')[-1]}")
```

## Submit to Leaderboard

Generate model card from results, then paste into your HF model's README.md header:
```bash
mteb create_meta --results_folder results/{model_name}/ --output_path model_card.md
```

The model appears on the [MTEB Chinese leaderboard](https://huggingface.co/spaces/mteb/leaderboard) after the daily refresh.
