# Evaluate on MIRACL (Condensed)

[MIRACL](https://project-miracl.github.io/) is a multilingual retrieval benchmark spanning 18 languages with native-speaker-generated topics and relevance labels.

## 1. Dataset

Dataset available on [HuggingFace](https://huggingface.co/datasets/miracl/miracl) with 16 "known" + 2 "surprise" languages. English alone has ~32.9M passages.

```python
from datasets import load_dataset

lang = "en"
corpus = load_dataset("miracl/miracl-corpus", lang, trust_remote_code=True)['train']
dev = load_dataset('miracl/miracl', lang, trust_remote_code=True)['dev']
```

**Corpus structure:** `docid` (format `x#y` where `x`=article ID, `y`=passage number), `title`, `text`.
**Dev structure:** `query_id`, `query`, `positive_passages`, `negative_passages` (same for train/dev/testA/testB).

```python
corpus_ids = corpus['docid']
corpus_text = [f"{doc['title']} {doc['text']}".strip() for doc in corpus]
queries_ids = dev['query_id']
queries_text = dev['query']
```

## 2. Evaluate from Scratch

### 2.1 Embedding

```python
from FlagEmbedding import FlagModel

model = FlagModel('BAAI/bge-base-en-v1.5')
queries_embeddings = model.encode_queries(queries_text)
corpus_embeddings = model.encode_corpus(corpus_text)
```

### 2.2 Indexing (Faiss)

```python
import faiss, numpy as np

dim = corpus_embeddings.shape[-1]  # 768 for bge-base-en-v1.5
index = faiss.index_factory(dim, 'Flat', faiss.METRIC_INNER_PRODUCT)
corpus_embeddings = corpus_embeddings.astype(np.float32)
index.train(corpus_embeddings)
index.add(corpus_embeddings)
```

### 2.3 Searching

```python
from tqdm import tqdm

all_scores, all_indices = [], []
for i in tqdm(range(0, len(queries_embeddings), 32), desc="Searching"):
    j = min(i + 32, len(queries_embeddings))
    score, indice = index.search(queries_embeddings[i:j].astype(np.float32), k=100)
    all_scores.append(score)
    all_indices.append(indice)

all_scores = np.concatenate(all_scores, axis=0)
all_indices = np.concatenate(all_indices, axis=0)

# Map results back to dataset IDs
results = {}
for idx, (scores, indices) in enumerate(zip(all_scores, all_indices)):
    results[queries_ids[idx]] = {
        corpus_ids[index]: float(score)
        for score, index in zip(scores, indices) if index != -1
    }
```

### 2.4 Evaluating

Download and parse qrels:

```python
file_name = "qrels.miracl-v1.0-en-dev.tsv"
endpoint = os.getenv('HF_ENDPOINT', 'https://huggingface.co')
os.system(f"wget {endpoint}/datasets/miracl/miracl/resolve/main/miracl-v1.0-en/qrels/{file_name}")

qrels_dict = {}
with open(file_name, "r", encoding="utf-8") as f:
    for line in f.readlines():
        qid, _, docid, rel = line.strip().split("\t")
        qrels_dict.setdefault(str(qid), {})[str(docid)] = int(rel)
```

Compute metrics with pytrec_eval:

```python
import pytrec_eval
from collections import defaultdict

ndcg_string = "ndcg_cut." + ",".join([str(k) for k in [10, 100]])
recall_string = "recall." + ",".join([str(k) for k in [10, 100]])

evaluator = pytrec_eval.RelevanceEvaluator(qrels_dict, {ndcg_string, recall_string})
scores = evaluator.evaluate(results)

all_ndcgs, all_recalls = defaultdict(list), defaultdict(list)
for query_id in scores:
    for k in [10, 100]:
        all_ndcgs[f"NDCG@{k}"].append(scores[query_id][f"ndcg_cut_{k}"])
        all_recalls[f"Recall@{k}"].append(scores[query_id][f"recall_{k}"])

for k in [10, 100]:
    all_ndcgs[f"NDCG@{k}"] = round(sum(all_ndcgs[f"NDCG@{k}"]) / len(scores), 5)
    all_recalls[f"Recall@{k}"] = round(sum(all_recalls[f"Recall@{k}"]) / len(scores), 5)
```

## 3. Evaluate using FlagEmbedding (Simplified)

Key parameters for the built-in evaluation runner:

```python
from transformers import HfArgumentParser
from FlagEmbedding.evaluation.miracl import MIRACLEvalArgs, MIRACLEvalModelArgs, MIRACLEvalRunner

# Key arguments:
#   --eval_name miracl  --dataset_names en  --splits dev
#   --search_top_k 100  --k_values 10 100
#   --eval_metrics ndcg_at_10 recall_at_100
#   --embedder_name_or_path BAAI/bge-base-en-v1.5
#   --devices cuda:0 cuda:1  --embedder_batch_size 1024

parser = HfArgumentParser((MIRACLEvalArgs, MIRACLEvalModelArgs))
eval_args, model_args = parser.parse_args_into_dataclasses()

runner = MIRACLEvalRunner(eval_args=eval_args, model_args=model_args)
runner.run()
```

Shell script also available at `examples/evaluation/miracl/eval_miracl.sh`. Results saved to `eval_results.json` under the configured output directory.
