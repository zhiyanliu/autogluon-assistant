# Condensed: Faiss GPU

Summary: This tutorial covers GPU-accelerated vector similarity search using Faiss, demonstrating how to transfer CPU indexes to GPU for faster indexing and searching. Key techniques include creating CPU indexes via direct initialization (`IndexFlatIP`) or `index_factory`, converting to GPU using `StandardGpuResources` with `index_cpu_to_gpu` (single GPU), `index_cpu_to_all_gpus` (all GPUs), and `index_cpu_to_gpu_multiple` (specific GPUs). It covers `GpuClonerOptions`/`GpuMultipleClonerOptions` for controlling GPU storage and shard/replicate behavior. Useful for implementing large-scale nearest neighbor search, embedding retrieval systems, and multi-GPU deployment of vector indexes.

*This is a condensed version that preserves essential implementation details and context.*

# Faiss GPU Tutorial (Condensed)

## Installation

GPU version only supports **Linux x86_64** via conda:

```
conda install -c pytorch -c nvidia faiss-gpu=1.8.0
```

## Data Preparation

```python
import faiss
import numpy as np

dim = 768
corpus_size = 1000
corpus = np.random.random((corpus_size, dim)).astype('float32')
```

## Create CPU Index

```python
# Direct initialization
index = faiss.IndexFlatIP(dim)

# Or via index_factory (composite indexes)
index = faiss.index_factory(dim, "Flat", faiss.METRIC_L2)
```

## GPU Index and Search

All GPU indexes require a `StandardGpuResources` object, which by default allocates **18% of total VRAM** as temporary scratch space. `GpuClonerOptions`/`GpuMultipleClonerOptions` are optional and control how objects are stored on GPU.

### Single GPU

```python
rs = faiss.StandardGpuResources()
co = faiss.GpuClonerOptions()
index_gpu = faiss.index_cpu_to_gpu(provider=rs, device=0, index=index, options=co)

index_gpu.add(corpus)
D, I = index_gpu.search(corpus, 4)
```

### All Available GPUs

Controls shard/replicate behavior via `GpuMultipleClonerOptions`:

```python
co = faiss.GpuMultipleClonerOptions()
index_gpu = faiss.index_cpu_to_all_gpus(index=index, co=co)
```

### Specific Multiple GPUs

```python
ngpu = 4
resources = [faiss.StandardGpuResources() for _ in range(ngpu)]

vres = faiss.GpuResourcesVector()
vdev = faiss.Int32Vector()
for i, res in zip(range(ngpu), resources):
    vdev.push_back(i)
    vres.push_back(res)
index_gpu = faiss.index_cpu_to_gpu_multiple(vres, vdev, index)
```

## Sanity Check

```python
# Nearest neighbor of each vector is itself
assert np.all(corpus[:] == corpus[I[:, 0]])
# Distances to self should be 0
print(D[:3])
```
