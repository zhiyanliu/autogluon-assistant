# Intro to Embedding — Condensed Tutorial

## Background

Traditional methods (one-hot, BoW, TF-IDF, BM25, n-grams) represent text as sparse vectors based on statistical features but suffer from high dimensionality and fail to capture semantic similarity (e.g., "cat" vs "kitty").

```python
# Bag-of-words example — sparse, no semantic meaning
words = ['I', 'love', 'basketball', 'have', 'a', 'match']
sen1_vec = [1, 1, 1, 0, 0, 0]  # "I love basketball"
sen2_vec = [1, 0, 1, 1, 1, 1]  # "I have a basketball match"
```

Dense embeddings (Word2Vec → RNNs/LSTMs → Transformers like BERT, GPT) map words to low-dimensional vectors capturing semantic relationships. **BGE (BAAI General Embedding)** is a key open-source model series.

```python
import os, numpy as np
os.environ['TRANSFORMERS_NO_ADVISORY_WARNINGS'] = 'true'
os.environ['CUDA_VISIBLE_DEVICES'] = '0'  # single GPU for small tasks

sentences = [
    "That is a happy dog",
    "That is a very happy person",
    "Today is a sunny day",
]
```

### Open-Source Models

**Advantages:** Free (check license), no rate limits, transparent/reproducible.

#### BGE
```python
from FlagEmbedding import FlagModel
model = FlagModel('BAAI/bge-base-en-v1.5')
embeddings = model.encode(sentences)
scores = embeddings @ embeddings.T  # cosine similarity via dot product
```

#### Sentence Transformers
```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer("all-MiniLM-L6-v2")
embeddings = model.encode(sentences, normalize_embeddings=True)
scores = embeddings @ embeddings.T
```

### Commercial Models

**Advantages:** No GPU needed, ecosystem integration, potentially better training data.

#### OpenAI
```python
os.environ["OPENAI_API_KEY"] = "YOUR_API_KEY"

from openai import OpenAI
client = OpenAI()
response = client.embeddings.create(input=sentences, model="text-embedding-3-small")
embeddings = np.asarray([response.data[i].embedding for i in range(len(sentences))])
scores = embeddings @ embeddings.T
```

#### Voyage AI
```python
os.environ["VOYAGE_API_KEY"] = "YOUR_API_KEY"

import voyageai
vo = voyageai.Client()
result = vo.embed(sentences, model="voyage-large-2-instruct")
embeddings = np.asarray(result.embeddings)
scores = embeddings @ embeddings.T
```

> **Key pattern:** All approaches produce embeddings where similarity is computed via matrix multiplication (`embeddings @ embeddings.T`). Normalize embeddings for proper cosine similarity.
