# Condensed: BGE Series

Summary: This tutorial covers implementing text embedding and similarity search using the BGE model family via the FlagEmbedding library. It demonstrates five model classes: `FlagModel` for BGE v1/v1.5 (BERT-based), `BGEM3FlagModel` for multilingual multi-functional embeddings (dense, sparse, ColBERT), `FlagLLMModel` for LLM-based multilingual embeddings (Gemma2), and `FlagICLModel` for in-context learning with few-shot examples (Mistral-7B). Key techniques include query instruction prepending, configuring `encode()` parameters (batch_size, max_length), enabling fp16, and computing similarity via dot product. Helps with semantic search, passage retrieval, and multilingual embedding tasks.

## BGE v1 & v1.5 (BERT-based)

**Models:** `large`/`base`/`small` variants for English and Chinese (33M–500M params). v1.5 improves similarity distribution and retrieval without instruction. Same API for both versions.

```python
from FlagEmbedding import FlagModel

model = FlagModel(
    'BAAI/bge-base-en-v1.5',  # or 'BAAI/bge-base-en' for v1
    query_instruction_for_retrieval="Represent this sentence for searching relevant passages:",
    query_instruction_format='{}{}'
)

q_embeddings = model.encode_queries(queries)      # prepends query_instruction to each query
p_embeddings = model.encode_corpus(corpus)         # calls encode() directly
scores = q_embeddings @ p_embeddings.T
```

**Key methods:** `encode(sentences, batch_size=256, max_length=512)` for general encoding; `encode_queries()` auto-prepends the retrieval instruction; `encode_corpus()` is an alias for `encode()`.

## BGE-M3 (XLM-RoBERTa-based, 568M params)

**Multi-Functionality** (dense, sparse, ColBERT), **Multi-Lingual** (100+ languages), **Multi-Granularity** (up to 8192 tokens).

```python
from FlagEmbedding import BGEM3FlagModel

model = BGEM3FlagModel('BAAI/bge-m3', use_fp16=True)

embeddings = model.encode(
    sentences,
    max_length=10,          # reduce from 8192 default to speed up
    return_dense=True,
    return_sparse=True,
    return_colbert_vecs=True
)
# Returns dict: {'dense_vecs': ..., 'lexical_weights': ..., 'colbert_vecs': ...}
# Each key is None if corresponding return flag is False
```

## BGE Multilingual Gemma2 (9.24B params, Gemma2-9B base)

LLM-based multilingual embedding model.

```python
from FlagEmbedding import FlagLLMModel

model = FlagLLMModel(
    'BAAI/bge-multilingual-gemma2',
    query_instruction_for_retrieval="Given a web search query, retrieve relevant passages that answer the query.",
    use_fp16=True  # speeds up with slight performance trade-off
)

embeddings_1 = model.encode_queries(queries)
embeddings_2 = model.encode_corpus(documents)
similarity = embeddings_1 @ embeddings_2.T
```

## BGE-ICL (7.11B params, Mistral-7B base, English)

Supports **in-context learning** via few-shot examples to enhance performance on new tasks.

```python
from FlagEmbedding import FlagICLModel

examples = [
    {
        'instruct': 'Given a web search query, retrieve relevant passages that answer the query.',
        'query': 'what is a virtual interface',
        'response': "A virtual interface is a software-defined abstraction..."
    },
    # additional examples...
]

model = FlagICLModel(
    'BAAI/bge-en-icl',
    examples_for_task=examples,  # set None to use without examples
    # examples_instruction_format="<instruct>{}\n<query>{}\n<response>{}"  # custom format
)

embeddings_1 = model.encode_queries(queries)
embeddings_2 = model.encode_corpus(documents)
similarity = embeddings_1 @ embeddings_2.T
```

**Key pattern across all models:** Encode queries and corpus separately, compute similarity via dot product (`q @ p.T`).
