## Architecture & Encoding

BGE-base-en-v1.5 uses **BERT-base** (12 encoder layers, hidden dim 768). BGE and BGE-v1.5 corresponding models share the same structure.

```python
from transformers import AutoTokenizer, AutoModel
import torch

tokenizer = AutoTokenizer.from_pretrained("BAAI/bge-base-en-v1.5")
model = AutoModel.from_pretrained("BAAI/bge-base-en-v1.5")
model.eval()
```

Tokenization uses `[CLS]` (101) and `[SEP]` (102) tokens, with `max_length=512`.

### Pooling

**⚠️ BGE uses `[CLS]` pooling, NOT mean pooling. Mean pooling causes significant performance degradation.**

```python
def pooling(last_hidden_state: torch.Tensor, pooling_method='cls', attention_mask: torch.Tensor = None):
    if pooling_method == 'cls':
        return last_hidden_state[:, 0]
    elif pooling_method == 'mean':
        s = torch.sum(last_hidden_state * attention_mask.unsqueeze(-1).float(), dim=1)
        d = attention_mask.sum(dim=1, keepdim=True).float()
        return s / d
```

### Complete Encoding Function

```python
def _encode(sentences, max_length=512, convert_to_numpy=True):
    input_was_string = False
    if isinstance(sentences, str):
        sentences = [sentences]
        input_was_string = True

    inputs = tokenizer(sentences, padding=True, truncation=True, return_tensors='pt', max_length=max_length)
    last_hidden_state = model(**inputs, return_dict=True).last_hidden_state
    embeddings = pooling(last_hidden_state, pooling_method='cls', attention_mask=inputs['attention_mask'])
    embeddings = torch.nn.functional.normalize(embeddings, dim=-1)

    if convert_to_numpy:
        embeddings = embeddings.detach().numpy()
    return embeddings[0] if input_was_string else embeddings
```

Key steps: tokenize → extract last hidden state → CLS pooling → **L2 normalize** → convert to numpy.

## FlagEmbedding API (Equivalent)

```python
from FlagEmbedding import FlagModel
model = FlagModel('BAAI/bge-base-en-v1.5')
embeddings = model.encode(sentences)
scores = embeddings @ embeddings.T
```

Both approaches produce identical results. The FlagEmbedding implementation adds batching, GPU support, and parallelization ([source](https://github.com/FlagOpen/FlagEmbedding/blob/master/FlagEmbedding/inference/embedder/encoder_only/base.py)).
