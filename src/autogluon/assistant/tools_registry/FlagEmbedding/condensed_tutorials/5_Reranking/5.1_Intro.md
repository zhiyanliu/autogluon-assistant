# Reranker Tutorial (Condensed)

**Concept:** Rerankers use cross-encoder architecture, taking query+text pairs simultaneously to score relevance directly. They're more accurate but slower than bi-encoders, so they're used as a second stage after initial retrieval.

## Setup & Data

```python
%pip install -U FlagEmbedding faiss-cpu

from datasets import load_dataset
import numpy as np

data = load_dataset("namespace-Pt/msmarco", split="dev")
queries = np.array(data[:100]["query"])
corpus = sum(data[:5000]["positive"], [])
```

## Embedding & Indexing

```python
from FlagEmbedding import FlagModel
import faiss

model = FlagModel('BAAI/bge-base-en-v1.5',
                  query_instruction_for_retrieval="Represent this sentence for searching relevant passages:",
                  use_fp16=True)
corpus_embeddings = model.encode(corpus)

dim = corpus_embeddings.shape[-1]
index = faiss.index_factory(dim, 'Flat', faiss.METRIC_INNER_PRODUCT)
corpus_embeddings = corpus_embeddings.astype(np.float32)
index.train(corpus_embeddings)
index.add(corpus_embeddings)
```

## Retrieval (Stage 1)

```python
query_embeddings = model.encode_queries(queries)
ground_truths = [d["positive"] for d in data]
corpus = np.asarray(corpus)

res_scores, res_ids, res_text = [], [], []
cut_offs = [1, 10]
k = max(cut_offs)

for i in range(0, len(query_embeddings), 256):
    q_embedding = query_embeddings[i: min(i+256, len(query_embeddings))].astype(np.float32)
    score, idx = index.search(q_embedding, k=k)
    res_scores += list(score)
    res_ids += list(idx)
    res_text += list(corpus[idx])
```

## Reranking (Stage 2)

**Available BGE Rerankers:**

| Model | Params | Base Model |
|:------|:------:|:----------:|
| `bge-reranker-v2-m3` | 568M | XLM-RoBERTa-Large |
| `bge-reranker-v2-gemma` | 2.51B | Gemma2-2B |
| `bge-reranker-v2-minicpm-layerwise` | 2.72B | MiniCPM (selectable output layers) |
| `bge-reranker-v2.5-gemma2-lightweight` | 9.24B | Gemma2-9B (selectable layers + compression) |
| `bge-reranker-large` | 560M | XLM-RoBERTa-Large |
| `bge-reranker-base` | 278M | XLM-RoBERTa-Base |

```python
from FlagEmbedding import FlagReranker

reranker = FlagReranker('BAAI/bge-reranker-large', use_fp16=True)
# use_fp16=True speeds up computation with slight performance degradation

# Basic usage: compute_score takes [query, text] pairs
scores = reranker.compute_score([
    ['what is panda?', 'Today is a sunny day'],
    ['what is panda?', 'The giant panda... is a bear species endemic to China.']
])
```

**Rerank retrieved results:**

```python
new_ids, new_scores, new_text = [], [], []
for i in range(len(queries)):
    new_score = reranker.compute_score([[queries[i], text] for text in res_text[i]])
    new_id = [tup[1] for tup in sorted(list(zip(new_score, res_ids[i])), reverse=True)]
    new_scores.append(sorted(new_score, reverse=True))
    new_ids.append(new_id)
    new_text.append(corpus[new_id])
```

## Evaluation (Recall, MRR, nDCG)

```python
def calc_recall(preds, truths, cutoffs):
    recalls = np.zeros(len(cutoffs))
    for text, truth in zip(preds, truths):
        for i, c in enumerate(cutoffs):
            recall = np.intersect1d(truth, text[:c])
            recalls[i] += len(recall) / max(min(len(recall), len(truth)), 1)
    return recalls / len(preds)

def MRR(preds, truth, cutoffs):
    mrr = [0 for _ in range(len(cutoffs))]
    for pred, t in zip(preds, truth):
        for i, c in enumerate(cutoffs):
            for j, p in enumerate(pred):
                if j < c and p in t:
                    mrr[i] += 1/(j+1)
                    break
    return [k/len(preds) for k in mrr]

# nDCG using sklearn
from sklearn.metrics import ndcg_score
pred_hard_encodings = [list(np.isin(pred, label).astype(int)) for pred, label in zip(new_text, ground_truths)]
for c in cut_offs:
    print(f"nDCG@{c}: {ndcg_score(pred_hard_encodings, new_scores, k=c)}")
```

**Key takeaway:** Compare metrics before/after reranking — reranking should improve recall, MRR, and nDCG, especially at small cutoffs like @1.
