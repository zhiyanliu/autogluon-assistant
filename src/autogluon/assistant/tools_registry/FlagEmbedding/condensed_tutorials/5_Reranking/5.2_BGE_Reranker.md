## Gen 1: bge-reranker (base/large)

Cross-encoder models (278M/560M params) based on XLM-RoBERTa, supporting Chinese and English. More accurate but less efficient.

```python
from FlagEmbedding import FlagReranker

model = FlagReranker('BAAI/bge-reranker-large', use_fp16=True, devices=["cuda:0"])

pairs = [
    ["What is the capital of France?", "Paris is the capital of France."],
    ["What is the capital of France?", "The population of China is over 1.4 billion people."],
]
scores = model.compute_score(pairs)
```

## Gen 2: bge-reranker-v2

Four models with different class imports, base models, and capabilities:

### v2-m3 (568M, XLM-RoBERTa-Large) — `FlagReranker`
Lightweight multilingual cross-encoder. Use `normalize=True` for 0-1 range via sigmoid.

```python
from FlagEmbedding import FlagReranker
reranker = FlagReranker('BAAI/bge-reranker-v2-m3', devices=["cuda:0"], use_fp16=True)
score = reranker.compute_score(['query', 'passage'], normalize=True)
```

### v2-gemma (2.51B, Gemma2-2B) — `FlagLLMReranker`
Strong English + multilingual performance.

```python
from FlagEmbedding import FlagLLMReranker
reranker = FlagLLMReranker('BAAI/bge-reranker-v2-gemma', devices=["cuda:0"], use_fp16=True)
score = reranker.compute_score(['query', 'passage'])
```

### v2-minicpm-layerwise (2.72B, MiniCPM) — `LayerWiseFlagLLMReranker`
Selectable output layers for accelerated inference. Strong English/Chinese.

```python
from FlagEmbedding import LayerWiseFlagLLMReranker
reranker = LayerWiseFlagLLMReranker('BAAI/bge-reranker-v2-minicpm-layerwise', devices=["cuda:0"], use_fp16=True)
score = reranker.compute_score(['query', 'passage'], cutoff_layers=[28])
```

### v2.5-gemma2-lightweight (9.24B, Gemma2-9B) — `LightWeightFlagLLMReranker`
Layerwise selection **plus** token compression for further resource savings.

```python
from FlagEmbedding import LightWeightFlagLLMReranker
reranker = LightWeightFlagLLMReranker('BAAI/bge-reranker-v2.5-gemma2-lightweight', devices=["cuda:0"], use_fp16=True)
score = reranker.compute_score(['query', 'passage'], cutoff_layers=[28], compress_ratio=2, compress_layers=[24, 40])
```

## Model Selection Guide

| Need | Recommended |
|------|------------|
| **Multilingual** | v2-m3, v2-gemma, v2.5-gemma2-lightweight |
| **Chinese/English** | v2-m3, v2-minicpm-layerwise |
| **Efficiency** | v2-m3, v2-minicpm-layerwise (low layers) |
| **Minimal resources** | bge-reranker-base/large |
| **Best performance** | v2-minicpm-layerwise, v2-gemma |

**Key notes:**
- `use_fp16=True` speeds up GPU computation with slight performance degradation
- Each model variant uses a **different class** (`FlagReranker`, `FlagLLMReranker`, `LayerWiseFlagLLMReranker`, `LightWeightFlagLLMReranker`)
- Use `devices=["cpu"]` if no GPU available
- Always benchmark on your actual use case for optimal speed-quality balance
