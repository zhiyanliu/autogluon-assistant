It covers computing binary relevance encodings via `np.isin`, set intersection with `np.intersect1d`, and building metric functions that operate on ranked result lists against ground-truth relevant document IDs. # Evaluation Metrics for Embedding Models

## Setup

```python
import numpy as np
from sklearn.metrics import ndcg_score

ground_truth = [
    [11,  1,  7, 17, 21],
    [ 4, 16,  1],
    [26, 10, 22,  8],
]
results = [
    [11,  1, 17,  7, 21,  8,  0, 28,  9, 20],
    [16,  1,  6, 18,  3,  4, 25, 19,  8, 14],
    [24, 10, 26,  2,  8, 28,  4, 23, 13, 21],
]
cutoffs = [1, 5, 10]

# Binary relevance encodings (needed for nDCG and MAP)
pred_hard_encodings = [list(np.isin(pred, label).astype(int)) 
                       for pred, label in zip(results, ground_truth)]
```

## 1. Recall

Ratio of relevant docs retrieved to `min(cutoff, total relevant docs)`:

$$\textbf{Recall}=\frac{|\text{Relevant} \cap \text{Retrieved}|}{\min(|\text{Retrieved}|, |\text{Relevant}|)}$$

```python
def calc_recall(preds, truths, cutoffs):
    recalls = np.zeros(len(cutoffs))
    for text, truth in zip(preds, truths):
        for i, c in enumerate(cutoffs):
            hits = np.intersect1d(truth, text[:c])
            recalls[i] += len(hits) / max(min(c, len(truth)), 1)
    recalls /= len(preds)
    return recalls
```

## 2. MRR (Mean Reciprocal Rank)

Averages `1/rank` of the **first** relevant result across queries:

$$MRR=\frac{1}{|Q|}\sum_{i=1}^{|Q|}\frac{1}{rank_i}$$

```python
def calc_MRR(preds, truth, cutoffs):
    mrr = [0 for _ in range(len(cutoffs))]
    for pred, t in zip(preds, truth):
        for i, c in enumerate(cutoffs):
            for j, p in enumerate(pred):
                if j < c and p in t:
                    mrr[i] += 1/(j+1)
                    break
    return [k/len(preds) for k in mrr]
```

## 3. nDCG (Normalized Discounted Cumulative Gain)

Measures ranking quality considering position via log-discount, normalized by ideal ranking (IDCG):

$$DCG_p=\sum_{i=1}^p\frac{2^{rel_i}-1}{\log_2(i+1)}, \quad nDCG_p=\frac{DCG_p}{IDCG_p}$$

```python
for c in cutoffs:
    print(f"nDCG@{c}: {ndcg_score(pred_hard_encodings, results, k=c)}")
```

## 4. Precision

Ratio of relevant docs retrieved to total docs retrieved at cutoff:

$$\textbf{Precision}=\frac{|\text{Relevant} \cap \text{Retrieved}|}{|\text{Retrieved}|}$$

```python
def calc_precision(preds, truths, cutoffs):
    prec = np.zeros(len(cutoffs))
    for text, truth in zip(preds, truths):
        for i, c in enumerate(cutoffs):
            hits = np.intersect1d(truth, text[:c])
            prec[i] += len(hits) / c
    prec /= len(preds)
    return prec
```

## 5. MAP (Mean Average Precision)

Average Precision per query: average precision at each relevant document's position. MAP averages AP across all queries.

$$\textbf{AP}=\frac{\sum_{k=1}^{M}\text{Relevance}(k) \times \text{Precision}(k)}{|\text{Relevant Docs}|}, \quad \textbf{MAP}=\frac{1}{N}\sum_{i=1}^{N}\text{AP}_i$$

```python
def calc_AP(encoding):
    rel, precs = 0, 0.0
    for k, hit in enumerate(encoding, start=1):
        if hit == 1:
            rel += 1
            precs += rel/k
    return 0 if rel == 0 else precs/rel

def calc_MAP(encodings, cutoffs):
    res = []
    for c in cutoffs:
        ap_sum = sum(calc_AP(enc[:c]) for enc in encodings)
        res.append(ap_sum/len(encodings))
    return res
```
